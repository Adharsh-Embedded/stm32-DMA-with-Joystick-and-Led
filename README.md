# STM32 DMA with Joystick and LED Control

## Project Overview
This project demonstrates joystick-controlled LED operation using the STM32F446RE microcontroller.

The joystick directions are read using ADC with DMA, and different LEDs are controlled based on joystick movement.

---

## Components Used
- STM32F446RE Nucleo Board
- Joystick Module
- LEDs
- Resistors
- Jumper Wires

---

## Features
- ADC with DMA
- Real-time joystick position reading
- LED control using joystick directions
- Efficient CPU usage using DMA

---

## Peripherals Used
- GPIO
- ADC
- DMA
- UART (optional for debugging)
